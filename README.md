## ✨ Características

- **Diseño Dark Mode Premium** — Paleta oscura con acentos índigo/violeta y efectos glassmorphism
- **Animaciones fluidas** — Scroll reveal, floating components y micro-interacciones en hover
- **Efecto Spotlight** — Las tarjetas reaccionan al movimiento del ratón con un efecto de luz
- **100% Vanilla** — Sin frameworks ni dependencias. Solo HTML, CSS y JavaScript puro
- **Totalmente Responsivo** — Adaptado para escritorio, tablet y móvil
- **SEO Ready** — Meta tags, Open Graph y estructura semántica correcta

## 🚀 Cómo correrlo localmente

```bash
# 1. Clona el repositorio
git clone https://github.com/Alekey-boop/portfolio.git

# 2. Entra a la carpeta
cd portfolio

# 3. Levanta un servidor local (requiere Python 3)
python3 -m http.server 8080

# 4. Abre en tu navegador
# http://localhost:8080
```

No se requieren dependencias de `npm`, instalaciones complicadas ni configuraciones. ¡Así de simple!
